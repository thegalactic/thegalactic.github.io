1 The Galactic Organization

Galactic stands for GAlois LAttices, Concept Theory, Implicational systems and Closures

The Galactic Organization maintains several java and php repositories:

  • java-lattices: a Java library for handling lattices
  • java-lattices-bin: a collection of Java command-line applications for handling lattices
  • php-packagist-keywords: an utility to extract a context from packagist.org from the most used keywords
